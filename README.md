# arduino-etch-sketch

#### Clone the repository:

```zsh
git clone https://github.com/username-projects/arduino-project.git
```

#### Install modules nodejs:

```zsh
cd ../arduino-project/
```

```zsh
npm install 
```
Create file `.env` and specify port.

#### Run app:

```zsh
cd ../server/
```
```zsh
node index.js /port-name/
```
#### Open in your browser:

```zsh
http://localhost:3000
```
